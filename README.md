# pony-music-lrcdb
A repository with (time synced) lyrics for pony music
Select a branch to view a specific artist/label

## Guidelines
- Naming scheme is as follows:
``album/albumname/01. songname.lrc``
``ep/albumname/01. songname.lrc``
(you get the point.... theres album, ep, compilation, single, remix... you get it)
- do not include things like (feat. ft.) etc.

## lyrics guidelines
- If theres a pause between lyrics... sync an empty line there with only a timestamp.
